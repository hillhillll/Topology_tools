Description
======
This is set of 
-> LAYNII programs  
-> ANTS alignement scripts
-> AFNI processing scripts
-> SPM motion correction scripts
-> ODIN programs
used for the mesoscale analysis of an fMRI study from Huber et al. 2019 
	https://www.biorxiv.org/content/10.1101/457002v2
